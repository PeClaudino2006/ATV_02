# 🛒 Lista de Compras - Atividade 02

Projeto desenvolvido para a disciplina **Desenvolvimento Web III** com o objetivo de criar uma **API de Lista de Compras** utilizando **Node.js, Express, TypeScript e MongoDB**.

A aplicação permite realizar operações de **CRUD (Create, Read, Update e Delete)** para gerenciar itens de uma lista de compras.

---

🚀 **Tecnologias**
- Node.js
- Express
- TypeScript
- MongoDB
- Mongoose
- MongoDB Compass

---

📦 **Instalação e Execução do Projeto**

Inicializar o projeto:

npm init -y

Instalar dependências:

npm install express mongoose cors body-parser

Instalar dependências de desenvolvimento:

npm install --save-dev typescript ts-node @types/express @types/node @types/mongoose @types/cors

Criar configuração do TypeScript:

npx tsc --init

Iniciar o servidor:

npm start

---

🗄️ **Banco de Dados**

Banco criado no **MongoDB Compass**

Database: shopping-list  
Collection: shoppingitems

---

⚙️ **Funcionalidades**

➕ Adicionar itens  
📋 Listar itens  
✏️ Atualizar itens  
❌ Excluir itens
